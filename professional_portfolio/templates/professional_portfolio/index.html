DCmd (Data Center Meta Data)


Description: A Data Information Management Web Application developed
for the University of Hawaii's Information Technology Center.

Goal: To create a centralized meta data information management system
and improve upon data access and data entry for the University's staff members.

Development Platform: Grails

Project Role/Duration: Developer, 2011-2014

Description: A music radio application that creates a customized music station

Development Platform: Appcelerator

Theme Park Scheduler Java Applet

Description: A Java applet that takes in a users selection of theme park attractions and

Development Platform: Java, Swing
